# YNBU3AWY Portfolio Website

Static bilingual Arabic/English portfolio website for GitHub Pages.

## Upload to GitHub Pages

1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. Go to Settings > Pages.
4. Under Build and deployment, choose: Deploy from a branch.
5. Select `main` branch and `/root`, then Save.

## Edit content

- Main text: `index.html`
- Styling: `assets/css/styles.css`
- Language text and interactions: `assets/js/main.js`
- Images: `assets/images` and `assets/thumbs`

## Notes

The site is fully static and uses no backend, so it is suitable for GitHub Pages.
